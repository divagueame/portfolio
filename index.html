IP Tracker

Maps/IP Based Location Tracker

'IP Tracker' is another project suggested in Front-end Masters. It utilizes two different APIs together: LeafletJS and the Geolocation API by IPfy. One of them is used to generate the map and the other one retrieves the IP location from the ISP. The user can enter any IP and the map will show the geolocation available.
